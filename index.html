Seguridad & Privacidad

ProtectID ha sido diseñado bajo la filosofía de cero transferencia de datos. Tus documentos personales nunca salen de este dispositivo.

🔒

Procesamiento 100% Local

Las imágenes se procesan en la memoria RAM del navegador. Funciona incluso si desconectas el internet por completo (Modo Avión).

🧼

Metadatos Eliminados (EXIF)

Al dibujar tu foto en el canvas del navegador, se elimina automáticamente toda la información oculta (coordenadas GPS, fecha y modelo de cámara).

🍪

Sin Cookies ni Rastreo

No guardamos datos de sesión ni instalamos cookies. Al recargar o cerrar la pestaña, todo rastro se destruye permanentemente.

🌐

Alojamiento CDN Seguro

Alojado de forma estática en GitHub Pages. Ningún servidor remoto procesa tus imágenes; el navegador se limita a descargar el código para ejecutarlo localmente.
